最近给自己立了几个 Flag，打算用技术解决一些生活中的实际问题，准备长期更新进度。

## 我想解决的问题
- 解决水课结课论文的自动化工作流。

- 实现 meetup host 填表单之后直接能够收到 email。

- 给我妈做一个八段锦打卡工具。   

## 工欲善其事，必先利其器
一开始，我打算试试 Gemini CLI，结果 sign in 好像有点问题，install 之后也一直在加载，不太行。

幸好社群里有朋友（min else）指点，发现问题可能是终端代理没配置好。我用的是 Clash for Windows，按照他的建议，在终端里 \`curl google.com\` 了一下，发现流量确实没走代理。解决方案是开启代理软件的 TUN 模式（也叫增强模式）。

捣鼓了蛮久，中间还跑偏去研究什么 API 登录，其实根本不是一回事。最后开了 TUN 模式就真的 OK 了！

果然是大家一起踩坑，每人踩几个，相互借鉴就不用全都踩了！💪😆

## 项目进展：东一锤，西一锤
工具搞定了，接下来就是项目的实际进展了。

**1. 论文自动化工作流**

我尝试用 RAG 技术来解决，但目前还没 figure out 一个可行的高效方法。这是一个暂时还 run 不动的 flow，令人痛苦面具。

**2. Meetup 自动化邮件**

这个我之前做过一个 flow，但后来琢磨了一下，发现像搞活动这种和现实场景、人密切相关的东西还是难以完全自动化。之后看看 AI Agent 进一步发展，也许还能实现帮忙根据活动城市找场地。✊

**3. 给妈妈的八段锦打卡 App**

这个项目最大的挑战是：我需要在 Windows 上开发一个 iOS app！😰

第一步就得先把 Mac 虚拟机装上。说实话，我一开始都不知道什么 Xcode 这些乱七八糟的🤯。搞了一整个下午，中间几乎没休息，下次得改一下，必须起来活动活动，实在是太上头了。

All in all，今天就是东一锤，西一锤，结果没有一个搞定。🤣
