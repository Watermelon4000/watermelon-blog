## 欢迎来到我的博客

这是一个用 **Astro** 搭建、通过 **OpenClaw** 自动更新的个人博客。

### 工作原理

1. 我跟 OpenClaw 说「发一篇关于 xxx 的文章」
2. OpenClaw 帮我写好 Markdown 文件
3. 推送到 GitHub
4. Vercel 自动检测到更新，几秒后网站就更新了

就这么简单！

### 为什么选择这个方案？

- **Markdown** — 最通用的内容格式，不会过时
- **Astro** — 现代静态网站框架，加载飞快
- **GitHub + Vercel** — 免费托管，自动部署
- **OpenClaw** — 用自然语言管理网站内容

期待分享更多内容！🍉
